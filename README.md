Chrome Flat CSS
===============

Chrome flat window in plain CSS

### Example usage

```html
<div class="chrome-flat">
    <ul>
        <li class="pinned"><a href="#">Pinned tab</a></li>
        <li><a href="#">Tab one</a></li>
        <li><a href="#">Tab with very long title</a></li>
        <li><a href="#">Tab four</a></li>
    </ul>
    <div>
        <h1>Hello world!</h1>
    </div>
</div>
```
